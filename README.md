# 从零搭建个人博客：WordPress建站完整指南

---

想拥有一个完全属于自己的网络空间？建立个人博客其实没有想象中那么复杂。这篇文章会带你一步步完成从域名注册到网站上线的全过程，用最直白的语言，帮你搞定技术细节。

---

## 开始之前，你需要准备这些

### 注册一个域名

域名就是你博客的"门牌号"。可以在阿里云、腾讯云、Godaddy、新网等平台注册。

挑域名有两个小技巧：
- 越短越好记（比如abc.com就比abcdefg.com好记）
- 优先选.com或.net这种常见后缀

### 买一台云服务器

这是你博客的"地基"。阿里云、腾讯云、华为云都有卖，如果服务器在国内就需要备案（大概一个月），选香港、台湾、澳门或其他海外地域就不用备案。

**一定要选CentOS或Ubuntu**，别选Windows——Windows吃资源太狠，跑个博客完全没必要。

## 关于备案这件事

买了国内服务器就得备案，流程很简单，主机商（阿里云、腾讯云等）都有备案系统，跟着步骤走就行。整个过程大概一个月，期间网站无法访问。

阿里云备案入口：https://beian.aliyun.com/

## 开始搭建：安装Web服务器

### 第一步：连接服务器

下载Xshell 6（解压后运行"!绿化.bat"就能用）。

打开软件，点"新建"：
- 主机：填你的服务器公网IP
- 端口：默认22
- 用户名：root
- 密码：你买服务器时设置的密码（忘了就去控制台重置）

![Xshell连接配置](image/96432610699777.webp)

![输入连接信息](image/11017141608065.webp)

![接受安全提示](image/24049147.webp)

### 第二步：装宝塔面板

连上服务器后，复制对应系统的命令粘贴进去：

**CentOS系统：**
```shell
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
```

**Ubuntu系统：**
```shell
wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

看到提示就输入"y"回车。

![安装宝塔面板](image/978121138.webp)

装完会显示登录地址、账号和密码：

![宝塔面板安装完成](image/43933091406.webp)

复制"外网面板地址"到浏览器打开。打不开？去安全组放行8888端口（百度搜"你的主机商+安全组放行端口"）。

进入面板后会弹出套件安装提示：
- PHP选7.1及以上版本
- Pure-Ftpd可以不装

点"一键安装"，等着就行。

![套件选择界面](image/14621964331139.webp)

![安装进度](image/652944608792393.webp)

装完记得在设置里改掉默认账号密码。

![修改面板密码](image/91636370337731.webp)

## 安装WordPress：两种方法任选

### 方法1：宝塔一键部署（推荐新手）

软件商店 → 一键部署 → WordPress → 填域名 → 选PHP版本 → 提交

![一键部署入口](image/119116230751196.webp)

别忘了去域名管理那里添加解析（@和www都解析到服务器IP）。

部署完成后浏览器输入域名访问，选简体中文继续。

![WordPress语言选择](image/6805619724.webp)

![填写数据库信息](image/1446138856.webp)

数据库信息用宝塔提供的，数据库地址和表前缀不用改。

![提交数据库配置](image/998719407.webp)

点提交，然后运行安装。

![开始安装WordPress](image/16876640.webp)

填网站名称、管理员账号密码、邮箱。**千万别勾"对搜索引擎不可见"**，勾了网站就没法被百度收录了。

![填写站点信息](image/679553561.webp)

装完访问域名就能看到你的博客了！后台地址：http://你的域名/wp-login.php

### 方法2：手动上传（想多了解细节可以试试）

点左侧"网站" → 添加站点 → 填域名（记得做DNS解析）

![添加网站](image/848807848.webp)

下载WordPress程序（最新版下载：https://imacro.lanzoui.com/ifLEGg2ogpi）

进入网站目录 → 上传zip文件 → 解压到当前目录

![上传WordPress](image/2045621746015157.webp)

![解压程序](image/768170007585562.webp)

![解压完成](image/767442734677243.webp)

![文件列表](image/230507630487.webp)

去"数据库" → 添加数据库 → 输入名称和密码

![创建数据库](image/0499723235932.webp)

![数据库信息](image/71508493628.webp)

浏览器访问域名，按刚才的步骤填数据库信息、站点信息，安装完成。

## 让博客更好看

WordPress自带的主题比较基础，可以去网上找更好看的。**建议用Github上的开源主题或付费主题**，安全性有保障。

支持PJAX的主题特别适合挂背景音乐——切换页面时不会中断播放。
